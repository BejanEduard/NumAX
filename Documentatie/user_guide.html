1. Introduction

1.1 Specificatii Proiect

Să se dezvolte o aplicație Web destinată colecționarilor de monede. Pe baza facilităților de căutare multi-criteriala implementate, utilizatorii autentificați vor putea crea, inventaria și partaja informații despre monede de interes în functie de diverse caracteristici (imagine, valoare, țară, perioadă de emisie, istoric etc.), plus vor putea importa/exporta datele referitoare la monedele deținute. Se vor genera statistici diverse ce pot fi exportate în formate deschise – minimal, CSV si PDF. Se va crea, de asemenea, un clasament al celor mai populare emisii monetare, disponibil și ca flux de date RSS. Inspiratie: Numista. Bonus: utilizarea datelor RDF oferite de nomista.org .

3. Utilizator delogat

3.1 Pagina de start (Home) :

Pe aceasta pagina, utilizatorul poate vedea informatii generale despre NumAX.

3.2 Catalog :

Pe aceasta pagina, utilizatorul poate vizualiza toate monedele existente in baza de date.

2.2 Filtrare :

In pagina de catalog este posibila filtrarea monedelor dupa propriile preferinte.

Aceste preferinte sunt de tipul:

  • Country
  • Composition
  • Shape

Dupa selectarea preferintelor, apasand pe butonul filter, pe pagina vor aparea un nou set de monede, filtrate dupa preferintele utilizatorului.

2.3 Register :

In pagina de register, un utilizator se poate inregistra pe site.

Pentru a reusi inregistrarea, utilizatorul trebuie sa completeze 4 campuri:

  • Username
  • Email
  • Password
  • Confirm Password

Exceptii

Daca cele doua parole nu coincid, un mesaj de eroare va fi afisat.

2.4 Login :

In pagina de login, un utilizator se poate loga pe site.

Pentru a reusi logarea, utilizatorul trebuie sa completeze 2 campuri:

  • Username
  • Password

Exceptii

Daca oricare dintre cele doua campuri nu coincid cu un utilizator creat anterior, va fi afisat un mesaj de eroare.

4. Utilizator Logat

4.1 Pagina de start (Home) :

See 3.1

4.2 Catalog

Langa fiecare moneda, va aparea un buton care va prezenta statusul monedei:

  • Add to your own collection - va adauga moneda la colectia personala
  • You already have this coin - moneda este deja prezenta in colectia personala

See 3.2

4.3 Filtrare

See 3.3

4.4 Create Coin :

Pe aceasta pagina, un utilizator va putea crea o moneda proprie. Dupa creare, moneda va fi adaugata automat in propria colectie.

Pentru a crea cu succes o moneda, trebuie completate 16 campuri de diferite tipuri:

  • Name
  • Period
  • In Circulation
  • Coin Description
  • Image 1
  • Image 2
  • Country
  • Value
  • Currency
  • Composition
  • Weight
  • Diameter
  • Thickness
  • Shape
  • Obverse description
  • Reverse description

Exceptii

Daca oricare dintre cele 16 campuri este completat gresit, sau o valoare lipseste, moneda nu va fi creata, ci in schimb vor fi afisate mesaje de tip eroare pentru fiecare camp gresit.

4.5 My Catalog :

Pe aceasta pagina se pot vizualiza toate monedele proprii.

In dreptul fiecarei monede, se afla un buton. Apasand pe acest buton, moneda va fi stearsa din colectia proprie, dar nu si din baza de date.

Apasand pe butonul "Export Your Coins", un document cu extensia .csv va fi descarcat pe calculatorul personal, continand informatii despre toate monedele din colectia proprie.

Apasand pe butonul "Import Your Coins", veti fi rugat sa atasati un document .csv valid.Dupa ce faceti acest lucru, fisierul cu monede incarcat de catre dvs. va fi incarcat in baza de date.

4.6 Statistics :

Pe aceasta pagina se pot vizualiza statistici despre monedele proprii.

Sunt disponibile 7 tipuri de statistici, anume:

  • Number of Coins by Country
  • Coins by Composition
  • Coins by Circulation
  • Coins by shape
  • Total weight
  • Average Diameter
  • Number of Coins

De asemenea, va fi disponibil un clasament al celor mai populare 5 emisii monetare din baza de date (dupa numarul de detinatori), NU din colectia proprie.

Acest clasament este disponibil si ca flux de date RSS, la adresa "rss.php"

Apasand pe butonul "Export Statistics to PDF", un document cu extensia .pdf va fi descarcat pe calculatorul personal, continand statistici despre toate monedele din colectia proprie.

Apasand pe butonul "Export Statistics to CSV", un document cu extensia .csv va fi descarcat pe calculatorul personal, continand statistici despre toate monedele din colectia proprie.

4.7 Logout :

Acest buton delogheaza utilizatorul.
